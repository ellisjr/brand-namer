# Brand Brief: Data Integration Middleware Platform

## What We're Naming
A data integration middleware platform — the company/product name for a Series B ($30M raised) startup that connects enterprise systems.

## What It Does
No-code visual builder with AI-powered data mapping that connects enterprise systems — Salesforce, Snowflake, Databricks, SAP — enabling interoperability across complex enterprise tech stacks. The core value proposition is being the "connective tissue" between systems.

## Who It's For
- **Primary:** Data engineering teams at enterprises running 50+ SaaS tools
- **Secondary:** IT leaders (VP/CIO-level) making platform purchasing decisions
- Developer-skewing audience that will judge the company on name quality

## Brand Personality Target
- Technical but accessible
- Reliable, enterprise-grade
- Competence + authority (Sage/Ruler archetype blend)
- Should feel peer-level alongside Salesforce, Snowflake, Databricks, AWS in a tech stack

## Competitive Landscape
| Competitor | Name Style | Energy |
|---|---|---|
| MuleSoft | Compound (animal + concept) | Workmanlike, technical |
| Boomi | Coined/sound-symbolic | Energetic, approachable |
| Fivetran | Portmanteau (five + transform) | Technical, precise |
| Airbyte | Compound (air + byte) | Technical, open-source energy |
| Workato | Blend (work + automate) | Functional, accessible |
| Tray.io | Found word + TLD | Simple, builder-energy |

## Integration Grammar Requirements
- "Powered by [Name]" — must carry authority
- "[Name] for Salesforce" — must feel like a peer product
- "[Name] + Snowflake" — must look natural in stack listings
- "[Name] Connect / [Name] API" — must extend cleanly

## The Name's #1 Job
Signal technical credibility to data engineers while feeling enterprise-grade to IT leaders. The name must survive the "would a VP put this on their LinkedIn" test AND the "would an engineer respect this" test.

## Prototype Targets (energy to aim at)
- **Stripe** — precision, engineer-respected, enterprise-trusted
- **Snowflake** — distinctive, concrete imagery, platform-scale
- **Confluent** — domain-relevant (confluence of data), technical vocabulary
- **Databricks** — compound that signals data + building
