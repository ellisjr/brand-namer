## Step 01b: Territories & Whitespace

### Competitive Whitespace Analysis

**Competitor naming patterns:**
| Competitor | Pattern | Category |
|---|---|---|
| MuleSoft | Animal metaphor (mule = beast of burden) | Metaphorical |
| Boomi | Sound-symbolic, playful, expansion | Sound-Symbolic |
| Fivetran | Technical compound abbreviation | Descriptive/Compound |
| Airbyte | Technical compound (air + byte) | Compound |
| Workato | Blend (work + auto) | Lexical Blend |
| Tray.io | Found word, minimalist | Found Word |

**Dominant patterns:** Technical/descriptive compounds, playful invented words, data-unit references
**Crowded territory:** -byte, -data, -flow, pipe/pipeline metaphors, cloud metaphors

**Whitespace opportunities identified:**
1. Precision/instrument vocabulary — no competitor uses measurement/calibration language
2. Orchestration/music metaphors — fresh territory for iPaaS
3. Infrastructure/fabric/textile metaphors — "connective tissue" angle unexplored
4. Cartography/navigation — data mapping as literal mapping
5. Classical/institutional authority — Latin/Greek roots for permanence

### Naming Territories

**Territory 1: Precision Instrument**
- Emotional promise: Engineered exactness, calibrated reliability
- Semantic field: gauge, caliber, true, mark, plumb, level, vernier, meridian, datum, axis
- Phonetic profile: Clean consonants, high front vowels (i, e), crisp endings. Competence dimension.
- Anti-patterns: Soft/warm, playful, consumer-friendly sounds

**Territory 2: Orchestration / Conductor**
- Emotional promise: Masterful coordination of many moving parts
- Semantic field: conductor, compose, chorus, cadence, score, tempo, fugue, harmony, baton, ensemble, opus
- Phonetic profile: Flowing liquids and fricatives, measured rhythm, 2-3 syllables
- Anti-patterns: Mechanical/industrial, aggressive, staccato

**Territory 3: Infrastructure Fabric**
- Emotional promise: The invisible but essential connective layer
- Semantic field: mesh, lattice, weave, fabric, span, truss, girder, rivet, stitch, splice, bond, sinew
- Phonetic profile: Strong consonant clusters, Anglo-Saxon feel, material/structural
- Anti-patterns: Abstract/ethereal, soft/decorative

**Territory 4: Cartography / Wayfinding**
- Emotional promise: Making complex landscapes navigable, data mapping as literal mapping
- Semantic field: traverse, bearing, azimuth, datum, contour, meridian, transit, vector, atlas, chart, course
- Phonetic profile: Directional energy, forward motion, crisp consonants
- Anti-patterns: Static, passive, contemplative

**Territory 5: Signal / Transmission**
- Emotional promise: Clean, reliable signal through noise — data moves where it should
- Semantic field: relay, signal, beacon, pulse, carrier, channel, band, trunk, circuit, switch, node
- Phonetic profile: Short, punchy, developer-credible, CLI-friendly
- Anti-patterns: Flowery, marketing-speak, verbose

**Session mode:** Explorer (detailed brief with clear taste signals)
